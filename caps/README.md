# MCCS Capabilities String Parser

[![release-badge][]][cargo] [![docs-badge][]][docs] [![license-badge][]][license]

`mccs-caps` provides a parser for the [MCCS](https://en.wikipedia.org/wiki/Monitor_Control_Command_Set)
capability string.
