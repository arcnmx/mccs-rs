# MCCS Database

[![release-badge][]][cargo] [![docs-badge][]][docs] [![license-badge][]][license]

`mccs-db` contains the human-readable descriptions of VCP features from the
[MCCS](https://en.wikipedia.org/wiki/Monitor_Control_Command_Set) spec.
